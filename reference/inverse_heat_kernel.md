# inverse_heat_kernel

inverse_heat_kernel

## Usage

``` r
inverse_heat_kernel(x, sigma = 1)
```

## Arguments

- x:

  the distances

- sigma:

  the bandwidth

## Value

Numeric vector/matrix of inverse heat kernel values.

## Examples

``` r
inverse_heat_kernel(c(1, 2), sigma = 1)
#> [1] 0.1353353 0.3678794
```
