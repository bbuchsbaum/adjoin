## Key Dependencies

**Critical:**
- Rcpp (>= 0.11.3) - C++ function binding via `@useDynLib neighborweights` in `R/neighborweights.R`
- RcppArmadillo - Linear algebra backend for weight calculations; referenced in `src/weight_funs.cpp` with `#include <RcppArmadillo.h>`
- Matrix - Sparse matrix (`dgCMatrix`, `sparseMatrix`) operations for adjacency matrices; used throughout `R/spatial_weights.R`, `R/diffusion.R`, `R/knn_weights.R`
- igraph - Graph representations; used in `R/neighbor_graph.R` for `igraph::graph_from_adjacency_matrix()`

**Nearest Neighbor Search:**
- RcppHNSW (>= 0.3.0.9001) - Hierarchical Navigable Small World approximate nearest neighbor search; primary NN library in `R/searcher.R`
- Rnanoflann - Fast exact nearest neighbor search via nanoflann; used in `R/spatial_weights.R` for radius-based searches
- FNN - Fast nearest neighbor implementation; fallback NN implementation

**Mathematics & Statistical:**
- RSpectra - Spectral decomposition for large matrices; used in `R/diffusion.R` for `eigs()` in eigendecomposition
- corpcor - Shrinkage-based covariance estimation; imported in `R/neighborweights.R`
- proxy - Distance and similarity computations; used via `proxy::simil()` in `R/neighborweights.R`
- mgcv - Generalized additive models for spatial autocorrelation; used in `R/spatial_weights.R` for `gam()` fitting

**Utilities:**
- assertthat - Input validation in `R/searcher.R` and throughout
- chk - Type checking in `R/searcher.R` via `chk_matrix()`, `chk_numeric()`
- furrr - Parallel mapping (future_map); imported in `R/neighborweights.R`
- crayon - Terminal colors for output formatting; imported in `R/neighborweights.R`
